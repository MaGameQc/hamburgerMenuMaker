Comment Ca Marche ?


Bienvenue sur BurgerMaker, un site web vous permettant de d'obtenir le code de l'icône hamburger de votre choix


1- pour commencer, cliquez sur les icônes pour trouver une animation qui vous plaît


2- Une fois que vous avez trouvé ce que vous cherchez, cliquez sur le bouton voir code


3- Un code HTML et CSS à été généré. il ne vous reste plus qu'à cliquer sur le bouton copier en bas du code généré


4- Allez dans votre fichier index.html, collez le code à l'endroit de votre choix, préférablement dans une barre de navigation de votre fichier HTML pour le code HTML, et dans votre fichier CSS pour le code CSS
